About Matemasie

Matemasie is a display typeface is inspired by Adinkra symbol Matemasie from the Akan people; an ethnic group from West Africa, primarily from Ghana and Cote d'Ivoire, with the name meaning “What I hear, I keep.” This symbol represents wisdom and knowledge.The Matemasie font is a one font style. It will have the Google Fonts GF Latin Core character set, as well as a handful of African-specific characters. It is anticipated to have about 330–350 glyphs, but might have more later for Google Fonts.
